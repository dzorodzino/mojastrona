link do ScreenShotów z Walidacji

1.Co to jest walidacja strony HTML? Walidacja strony HTML to proces sprawdzania kodu HTML strony internetowej pod kątem zgodności ze standardami ustalonymi przez organizacje takie jak W3C (World Wide Web Consortium). Co to oznacza w praktyce? Podczas walidacji analizuje się, czy kod HTML jest: poprawnie napisany składniowo (np. czy wszystkie tagi są poprawnie otwarte i zamknięte), zgodny ze specyfikacją wybranej wersji HTML (np. HTML5, XHTML), wolny od błędów, które mogą utrudniać wyświetlanie strony w różnych przeglądarkach.

2.Czym jest walidator? Walidator to narzędzie, które automatycznie sprawdza poprawność kodu (np. HTML, CSS, XML) względem określonych reguł, standardów lub schematów. W kontekście stron internetowych: Walidator HTML Sprawdza, czy kod HTML: jest poprawny składniowo, jest zgodny z wybraną wersją HTML (np. HTML5), nie zawiera błędów (np. brakujących tagów, złego zagnieżdżenia). Walidator CSS Sprawdza poprawność stylów CSS, np.: czy właściwości są poprawnie zapisane, czy wartości są prawidłowe (np. color: red zamiast color: 123).

Możliwość formatowania tekstu w HTML.
  • Pogrubienie z podkreśleniem ważności: < strong > < /strong >
  • Kursywa z naciskiem (ważność): < em > < /em >
  • Wyróżnione: < mark > < /mark >

  • Mały tekst: < small > < /small >
  • skreślony tekst: < del > < /del >
  • tekst dodany (podkreślony): < ins > < /ins >

  • tekst z klawiatury: < kbd > < /kbd >
  • odwrócenie kierunku tekstu: < bdo > < /bdo >
  • Cytat blokowy: < blockquote > < /blockquote >
